# 📝 Todo List Dashboard 사용 안내

이 폴더는 사용자님의 업무 관리 대시보드 공식 작업 공간입니다.

## 🚀 시작하기 (실행 방법)
1. 폴더 안에 있는 **`실행하기.bat`** 파일을 더블 클릭합니다.
2. 검은색 터미널 창이 뜨면서 서버가 시작됩니다.
3. 브라우저 주소창에 `http://localhost:3000`을 입력하고 접속합니다.

## 🛠 코드 수정하기
- **VS Code**에서 이 폴더를 열어서 `index.html`, `style.css`, `app.js` 등을 수정하실 수 있습니다.

## 📁 주요 파일 설명
- `server.js`: 데이터를 저장하고 불러오는 서버 역할을 합니다.
- `index.html`: 화면의 뼈대를 담당합니다.
- `style.css`: 화면의 디자인을 담당합니다.
- `app.js`: 버튼 클릭 등 모든 기능을 담당합니다.
- `data.json`: 사용자님이 입력한 업무 데이터가 저장되는 파일입니다. (지우지 마세요!)
